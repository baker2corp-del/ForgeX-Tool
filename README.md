## ForgeX-Tool

## Overview

ForgeX-Tool is a lightweight Discord multitool designed to give users a quick, visual overview of key server functions. It is strictly a read-only viewer, meaning you can see everything without making changes. Perfect for monitoring or exploring Discord features safely.

## Features
# 1. Banner

View banners displayed in the tool.

Helps identify server sections or highlight specific content.

# 2. Menu

Access and view the main navigation menu.

Shows available options and commands clearly.

# 3. User Input

See what commands or data have been entered.

Useful for monitoring actions without interfering.

# 4. Webhook Function

View messages sent via Discord webhooks.

Keep track of automated notifications or announcements.

# 5. Colors

See text displayed in different colors.

Makes reading and differentiating information easier and more visually appealing.

## Usage

Run the tool and follow the on-screen menu to navigate.

All features are view-only; no modifications are possible.

Ideal for monitoring server activity and exploring tool functions.
